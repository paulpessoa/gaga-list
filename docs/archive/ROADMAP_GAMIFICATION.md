# Roadmap de Gamificação e Crescimento: Grãos IA 🚀

Este documento registra as ideias estratégicas para manter o app viral, rentável e engajador, conforme discutido entre o Founder e o Staff Engineer.

## 1. Estratégias de Aquisição e Viralidade
- **Programa de Indicação (Member Get Member):** 
  - Cada usuário tem um código/link único.
  - Ao convidar um amigo que cria sua primeira lista, ambos ganham 50 grãos.
- **Viralidade via Compartilhamento:** 
  - Ao compartilhar uma lista que atinge 3 colaboradores ativos, o dono da lista ganha um bônus de grãos.

## 2. Gamificação e Retenção (Earn Grains)
- **Daily Check-in:** Abra o app 3 dias seguidos e ganhe 10 grãos.
- **Tarefas de Compras:** 
  - Marcar 10 itens como comprados: +5 grãos.
  - Usar o GPS/Radar em conjunto: +10 grãos para o grupo.
- **Anúncios Premiados (Ads):** 
  - Assistir a um vídeo curto (30s) para recarregar 20 grãos (ideal para usuários do plano grátis).

## 3. Transparência e Monetização
- **Token Awareness:** Exibir claramente o custo de cada ação de IA na interface (Implementado ✅).
- **Planos vitalícios (Lifetime):** Considerar um plano de "Grãos Infinitos" para usuários Early Adopters.

## 4. Admin Financeiro e Operações
- **Controle de Margem:** Monitorar o custo real de cada API (Gemini, OpenAI, Groq) vs o valor pago pelo usuário.
- **Auditoria:** Lista de transações do Stripe integrada ao painel staff para estornos e suporte.
